# myblogs

Personal blog built with Next.js, TypeScript, and Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Scripts

- `npm run dev` — start the dev server
- `npm run build` — production build
- `npm run start` — start the production server

## Deploy on Vercel

This is ready for Vercel (Next.js). You can connect the GitHub repo and deploy with defaults.

If you want me to configure Vercel via CLI, I can set it up for you.
