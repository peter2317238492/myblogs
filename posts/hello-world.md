# 欢迎 👋

很高兴你找到这里！这是一个用 **Next.js** + **Tailwind CSS** 构建的技术博客。

## 特性

这个博客具备以下特性：

- ✨ **现代化技术栈** - Next.js 14 + TypeScript + Tailwind CSS
- 🌙 **深色模式** - 支持浅色/深色模式自动切换
- 🔍 **全文搜索** - 快速搜索文章
- 🏷️ **标签分类** - 按标签浏览文章
- 📱 **响应式设计** - 完美适配各种设备
- 🚀 **高性能** - 静态生成，秒级加载

## 使用方法

### 1. 添加新文章

在 `posts` 目录下创建新的 Markdown 文件：

\`\`\`markdown
---
title: "文章标题"
date: "2026-01-29"
excerpt: "文章摘要"
tags: ["标签1", "标签2"]
---

# 文章内容

在这里写你的文章内容...
\`\`\`

### 2. 启动开发服务器

\`\`\`bash
npm run dev
\`\`\`

访问 http://localhost:3000 查看效果。

### 3. 部署

将项目推送到 GitHub，然后在 Vercel 上导入即可自动部署。
