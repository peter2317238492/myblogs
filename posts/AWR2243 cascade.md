# AWR2243级联板

## Table 1. 关键特性（Key Features）

### Cascade Radar RF Board

| 项目                       | 说明                                                         |
| -------------------------- | ------------------------------------------------------------ |
| 4× AWRx 76–81 GHz 雷达 SoC | 集成 VCO、LO 分配、PA、LNA、ADC；3 TX、4 RX；ARM MCU R4 控制器 |

---

### AWRx RF / LO 外设

| 项目                          | 说明                                                         |
| ----------------------------- | ------------------------------------------------------------ |
| 12× TX，16× RX 天线           | - 4 个 AWRx 设备共 12 个发射器（TX）<br>- 4 个 AWRx 设备共 16 个接收器（RX）<br>- 86 个不重叠的方位向虚拟阵列（λ/2）<br>- 4 层最小冗余阵列（MRA）俯仰方向 |
| 嵌入式天线                    | - 4 单元串馈贴片天线，12 dBi<br>- 方位角 ±60°（3 dB）<br>- 俯仰角 ±30°（3 dB） |
| 20 GHz 本振（LO）星型网络分配 | 由主 AWRx 设备的 LO 输出驱动，通过 2 个 Wilkinson 功分器向主/从 AWRx 设备分配 |

---

### AWR 数字外设（Digital Peripherals）

| 项目           | 说明                                             |
| -------------- | ------------------------------------------------ |
| CSI-2.0 四通道 | 600 Mbps/通道，每设备 ADC IF 数据速率达 2.4 Gbps |
| QSPI Flash     | 16 Mbit QSPI Flash，用于 AWR 固件更新/开发       |
| 串行外设       | SPI、I2C、UART、GPIO                             |
| 系统温度       | TMP112 I2C 温度传感器                            |

---

### 电源（Power）

| 项目                        | 说明                                                         |
| --------------------------- | ------------------------------------------------------------ |
| 雷达电源管理 IC（PMIC）方案 | - 2× LP87524P 四通道集成 FET 降压转换器<br>- 二级 LC 滤波方案，为 4 个 AWRx 设备供电<br>- TPS73733 LDO 生成 3.3 V 系统电源 |
